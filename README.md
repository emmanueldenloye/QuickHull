# QuickHull

This just contains the 2D implementation. I'll attempt the general case at a later time.

# Dependencies

## Ubuntu / Debian

```
$ sudo apt-get build-dep cairo
$ sudo apt-get install libglib2.0-dev libcairo2-dev libpango1.0-dev
```

# Building

```
$ stack build
```

# Running tests

```
$ stack test
```
